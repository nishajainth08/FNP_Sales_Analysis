FNP Sales Analysis Dashboard

Overview
This project is an Excel-based dashboard analyzing sales performance using Power Pivot and Data Model.

* Key Metrics
- Total Orders: 1000
- Total Revenue: ₹35,20,984
- Avg Order to Delivery Time: 5.53 days
- Avg Customer Spend: ₹3,520

* Insights
- Peak revenue during major occasions like Anniversary & Raksha Bandhan
- Certain product categories outperform consistently
- Delivery time impacts customer spending behavior

* Tools Used
- Microsoft Excel
- Power Pivot
- Data Model
- Pivot Tables & Charts

* Files Included
- Excel Dashboard (.xlsx)
- Raw datasets (.csv)
- Dashboard screenshot
